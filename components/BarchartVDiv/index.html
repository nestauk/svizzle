Basic props

In the most basic setup, you need to provide a `{items}` array of objects with the shape `{key: string, props: number}`.

Note that if there are both positive and negative values the chart will show a vertical axis, `grey` by default.

If `items` is undefined or empty the chart shows a message that you can customize using the props `message`, `theme.messageColor` (default: black) and `theme.messageFontSize` (default: 1rem).
