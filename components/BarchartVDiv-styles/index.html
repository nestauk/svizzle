Styles

You can setup a `backgroundColor` and the `itemTextColorDefault`.

`barHeight` and `fontSize` contribute to determine the distance between bars.

You can configure the axis color using the `axisColor` props (used in case there are values of both signs).

You can choose the hovered bar background color by providing `theme.itemBackgroundColorHover`.

You can set the hovered bar color by providing `theme.itemBarColorHover` (`null` by default, which does not affect the bar color).

Choose an example

Usage

<BarchartVDiv
  {items}
  barHeight = 12
  theme={{
    axisColor: 'red',
    backgroundColor: 'antiquewhite',
    fontSize: 22,
    itemBackgroundColorHover: 'cyan',
    itemBarColorHover: 'red',
    itemTextColorDefault: 'green',
    itemTextColorHover: 'orange',
  }}
/>
