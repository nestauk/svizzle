Reference lines

You can show reference lines by providing an array `refs` with defaut shape `{key: string, value: number}[]`.

Reference items can also have props:

• `keyAbbr`, the string to use if the ref label for that item overflows the barchart width

• `format`, a function to format the value to be shown in the ref label

You can style reference lines usind the `theme` props `refColor`, `refDasharray` and `refWidth`.

Choose an example

Usage

<BarchartVDiv
  {items}
  refs={[
    {key: 'National average', value: 200}
  ]}
/>
